1. 템플릿 엔진

    템플릿 양식과 특정 데이털 모델에 따른 입력 자료를 합성하여
    결과 문서(응답화면)를 출력하는 것
    -> 만들어둔 화면(html)에 데이터를 추가하여 하나의 html로 만들어서 응답
    (JSP도 템플릿 엔진 중 하나임!)
    타임리프 홈페이지

2. Thymeleaf

    웹 및 독립 실행형 환경을 모두 지원하는 최신 서버 측 Java 템플릿 엔진
    -> 웹 실행 : 요청 시 포워드 되는 화면
    -> 독립 실행 : html 파일만 실행

    HTML 파일에서 th(thymeleaf) 속성을 이용해 컨트롤러로부터 전달받은 데이터를
    이용해 동적 페이지를 만들 수 있음

    Spring Boot 에서는 JSP가 아닌 Thymeleaf 사용을 권장하고 있음.
