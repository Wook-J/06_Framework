조회수를 증가시키는 방법
1. 조회할 때 마다 증가
2. DB에 누가 어떤 글을 조회했는가를 일정 기간 단위로 확인해서 증가
   (if 회원 1번이 2003번 게시글을 241120에 조회 -> 하루단위로 정해서 증가여부 결정!)
3. local/session 스토리지(JS에서 사용가능, 서버에서는 사용 불가능... -> 잘 안씀)
4. 쿠키를 이용한 조회 수 증가
5. In Memory DB (나중에 시간되면 도전..?) (redis....?? + AWS...??)
   HDD, SDD 에 저장하는 것이 아닌 ""메모리""에 데이터 저장


쿠키 -> 서버에서 생성 -> 클라이언트에게 전달
생성된 쿠키를 전달받은 클라이언트가 쿠키를 저장/관리
(클라이언트의 조작의 가능성이 높음!!!)

조회수 증가 규칙
1. 오늘 읽은 글을 또 조회수 증가 X
2. 글쓴이는 조회수 증가 X
